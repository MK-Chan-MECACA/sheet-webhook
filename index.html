Setup

Live in about 3 minutes

Do this once per sheet. No add-ons to install, no servers, no monthly fees.

1

Open Apps Script

In your Google Sheet, go to Extensions ▸ Apps Script.

2

Paste the code

  1. Replace everything in Code.gs with the Code.gs below.
  2. Click + ▸ HTML, name it exactly Sidebar, paste the Sidebar.html.
  3. (Recommended) Show appsscript.json in Project Settings and paste it too.
  4. Hit Save 💾, then reload the sheet — a 🔗 Webhook menu appears.
⚡ One command instead? If you have Node, run clasp create --type sheets then clasp push. See the README on GitHub.
3

Deploy as a Web App (one time)

  1. Deploy ▸ New deployment → gear ⚙ → Web app
  2. Execute as: Me  ·  Who has access: Anyone
  3. Deploy → Authorize access → allow
This is the only manual step Google requires. The sidebar walks you through it.
4

Copy your webhook URL

Open 🔗 Webhook ▸ Get Webhook URL, click 📋 Copy URL, then ⚡ Send Test Row to confirm it works.

5

Paste it into your form

Drop the URL into your form's webhook / "POST to URL" field (Typeform, Jotform, Tally, Webflow…). Submit — the row shows up instantly.

Built in

Simple, but not basic

🧱

Auto headers

The first submission builds the header row from your field names. New fields later? They add new columns on their own.

🔒

Secret key

Flip on protection in the sidebar — a private token is added to the URL and any request without it is rejected.

🧩

Field mapping

Rename, reorder, or drop columns from a simple config tab. No code — and nothing listed there is ever lost.
